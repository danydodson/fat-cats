Some cats are born fat, some achieve fatness, others have fatness thrust upon them. You can find all of them at the San Francisco SPCA using this tool.

## Install

Compatible with node 6+

`npm install -g fat-cats`

## Usage

##### Standard
`$ fat-cats`

##### Metric units
`$ fat-cats --metric`

##### Garfield Mode
`$ fat-cats --garfield`
